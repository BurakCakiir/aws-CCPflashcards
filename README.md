# AWS Flashcards

AWS sertifika sınavlarına hazırlık için interaktif flashcard uygulaması.

## Nasıl Kullanılır?

1. `index.html` dosyasını indirin
2. Tarayıcınızda açın (Chrome, Firefox, Edge vb.)
3. Kartlara tıklayarak soruları ve cevapları görün

Başka bir kurulum veya programa gerek yoktur. İnternet bağlantısı sadece fontlar için gereklidir.

## Özellikler

- Kartları çevirerek soru/cevap görme
- Kategori bazlı filtreleme
- Mobil uyumlu tasarım



# AWS Flashcards

An interactive flashcard application for preparing for AWS certification exams.

## How to Use?

1. Download the `index.html` file.
2. Open it in your browser (Chrome, Firefox, Edge, etc.).
3. Click on the cards to see the questions and answers.

No other installation or programs are needed. An internet connection is only required for the fonts.

## Features

- View questions/answers by flipping cards.
- Category-based filtering.
- Mobile-friendly design.
